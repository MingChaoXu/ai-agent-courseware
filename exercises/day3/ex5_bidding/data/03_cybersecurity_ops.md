## 第三部分 项目需求

### 3.1 项目背景

S市电子政务外网承载全市120余个政务信息系统，涉及60余家市直单位。近年来网络安全形势日趋严峻，多次受到境外APT攻击和勒索病毒威胁，急需专业安全运维服务保障政务系统安全稳定运行。

### 3.2 服务内容

1. **日常安全监测**：7×24小时安全事件监测与研判，覆盖网络边界、主机系统、应用系统、数据安全；
2. **漏洞管理**：每月进行1次全面漏洞扫描，每季度进行1次渗透测试，出具风险评估报告并跟踪整改；
3. **应急响应**：安全事件发生后2小时内到场、4小时内出具初步分析报告、24小时内完成处置；
4. **安全巡检**：每周对核心网络设备、安全设备进行1次安全配置审计；
5. **安全培训**：每季度开展1次全市政务系统安全意识培训，全年不少于4次；
6. **重保支撑**：在国家重大活动期间提供驻场安全保障服务；
7. **等保合规**：协助完成等级保护测评整改，确保系统符合等保2.0标准。

### 3.3 服务要求

1. 服务期限为1年，期满经考核合格可续签1年（最多续签1次）；
2. 驻场运维人员不少于3人，其中安全分析师不少于1人；
3. 所有安全事件需在24小时内上报主管部门；
4. 不得将项目转包或将核心服务内容分包；
5. 服务期内如发生重大安全责任事故，采购人有权单方解除合同。

### 3.4 交付成果

1. 月度安全运维报告；
2. 季度安全评估报告；
3. 安全事件分析报告（按次）；
4. 年度安全总结报告；
5. 漏洞扫描与渗透测试报告。
