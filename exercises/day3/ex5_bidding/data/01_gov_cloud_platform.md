## 第三部分 项目需求

### 3.1 项目背景

W市现有政务云平台于2022年建成投入使用，随着政务数字化程度不断加深，现有平台在算力、存储、安全防护等方面已无法满足需求。本项目旨在对现有政务云平台进行扩容升级及安全加固。

### 3.2 建设内容

1. **计算资源扩容**：新增物理服务器20台，虚拟化平台授权扩展，满足200个政务应用部署需求；
2. **存储资源扩容**：新增分布式存储容量200TB，满足未来3年政务数据存储需求；
3. **网络安全加固**：部署下一代防火墙、Web应用防火墙、数据库审计系统、堡垒机等安全设备；
4. **容灾备份建设**：建设同城灾备中心，实现RPO≤1小时、RTO≤4小时；
5. **运维管理平台**：建设统一运维监控平台，实现对计算、存储、网络、安全资源的统一监控和管理。

### 3.3 技术要求

1. 服务器采用国产化CPU（鲲鹏或海光），支持信创生态；
2. 虚拟化平台支持VMware vSphere及国产化HyperCloud双栈；
3. 存储系统须支持SSD+HDD混闪，提供不少于3份数据副本；
4. 安全设备须具备国家信息安全测评认证证书；
5. 监控平台须提供API接口，支持与省级政务云监管平台对接；
6. 全部系统支持IPv4/IPv6双栈。

### 3.4 交付要求

1. 签订合同后90个日历天内完成全部设备到货及安装调试；
2. 签订合同后120个日历天内完成系统上线并通过验收；
3. 提供至少3年7×24小时原厂质保及运维服务。
